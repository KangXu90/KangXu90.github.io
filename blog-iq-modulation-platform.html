Building a Low-cost, High-performance IQ Modulation Platform for NV-center Quantum Experiments

Published July 31, 2025 · Adapted from my LinkedIn article

Precise microwave phase control is central to NV-center quantum sensing. This note describes a practical IQ modulation platform assembled from affordable second-hand instruments and integrated into a laboratory pulse-control workflow.

Motivation

Quantum sensing and magnetic resonance experiments with nitrogen-vacancy centers in diamond often require clean microwave phase control. A common route is to use a modern vector signal generator or a high-end AWG, but that can be expensive when the experiment only needs a compact set of stable phase states.

In our lab, we tested a more economical approach built around a Tektronix AWG2005 arbitrary waveform generator and an Agilent E4433B ESG signal generator. Both instruments are widely available on the second-hand market, and together they can provide reliable IQ control at a fraction of the cost of a new integrated microwave-control platform.

IQ Modulation Strategy

The basic idea is to use two orthogonal control channels: I for the in-phase component and Q for the quadrature component. By choosing the proper I/Q voltage pair, the microwave carrier phase can be mapped to the phase states needed for pulse sequences such as Rabi, Ramsey, and Hahn echo experiments.

The AWG2005 was operated in DC mode to provide stable and symmetric bias voltages for the I and Q inputs of the E4433B. In this configuration, the AWG functions less like a waveform synthesizer and more like a precise, programmable phase-state source.

Hardware Setup

The working setup used a few practical building blocks:

  • Tektronix AWG2005 for stable DC bias outputs on the I/Q channels.
  • Agilent E4433B ESG signal generator, covering the microwave frequencies needed for NV-center work.
  • Matched microwave switches for the I and Q paths to preserve timing and amplitude symmetry.
  • SpinCore PulseBlaster TTL outputs to coordinate phase selection, microwave gates, laser timing, and acquisition windows.

A small but important detail is timing margin. The I/Q control pulses should start before the microwave pulse and end after it, so the microwave burst occurs only while the selected phase state is already stable. Oscilloscope checks at each stage are essential; they catch timing skew, pulse-width mistakes, and channel asymmetry before those issues appear as confusing quantum-control data.

Software Integration

The control layer was integrated into existing laboratory software based on ImageAcquisition and NVCommandCenter workflows. The main addition was custom IQ phase-sequence logic inside the pulse-sequence sweep module, so phase choices could be generated programmatically rather than patched manually into each experiment.

Instrument driver functions were also adapted for the E4433B and for the IQ modulation workflow. This made it easier to combine microwave frequency scans, phase cycling, pulse-length optimization, laser timing, and APD acquisition delays in a repeatable way.

Experimental Validation

We validated the platform through several standard NV-center experiments. First, ODMR scans showed clear resonance features, including resolved hyperfine structure from nitrogen. Then, after optimizing microwave pulse duration, laser timing, and readout windows, Rabi oscillations confirmed stable coherent driving.

Finally, Hahn echo measurements demonstrated reliable phase shifts for the classical pi/2 - pi - pi/2 readout sequence. The observed echo response, including carbon-13 induced revivals, confirmed that the system could support real quantum-sensing measurements rather than only bench-level microwave tests.

Practical Lessons

  • Use identical microwave switches for the I and Q channels whenever possible.
  • Give the I/Q phase-selection pulses enough timing margin around the gated microwave pulse.
  • Verify every timing assumption on an oscilloscope before interpreting quantum data.
  • Optimize microwave, laser, and APD windows systematically rather than changing several parameters at once.
  • Keep phase-sequence generation inside the control software so the experiment remains reproducible.

Cost and Value

The core second-hand instruments cost roughly $1500 in total, and the full IQ modulation path can be built for around $2000 excluding the PulseBlaster. For many labs, that is an attractive middle ground: much more capable than improvised manual phase control, but far less expensive than a new high-end vector microwave platform.

Conclusion

This project shows that precise phase control for NV-center experiments does not always require a costly all-in-one instrument. With careful hardware matching, timing verification, and software integration, a low-cost IQ modulation system can support high-quality quantum-control experiments.
